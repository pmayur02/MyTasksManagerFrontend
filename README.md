# React Todo App Frontend

This is the frontend for the Todo App, built with **React**.  
It consumes the FastAPI backend API for authentication and task management.

---

## Features

- User login and registration
- Add, edit, delete, and view tasks
- Task status: `pending` or `completed`
- Responsive design
- Environment variables support for API URL

---

## Tech Stack

- React 18+
- React Router DOM
- Axios
- CSS (modular styles)
- Environment variables (`.env`)

---

## Setup

1. Clone the repo:

2. Install dependencies:
npm install

3. Create .env file:
VITE_BACKEND_URL = Your Backend URL

4. Run development server:
npm run dev
